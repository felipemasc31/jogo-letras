# Jogo das Letras

Letras caem do topo da tela e o jogador deve pressionar a tecla para eliminá-las e ganhar pontos.
Se uma letra chegar ao chão o jogador perde uma vida. O jogo termina quando todas as vidas acabam.

# Funcionalidades

- **Letras caem aleatoriamente**
- **Detecta teclas pressionadas**
- **Sistema de pontuação** (+10 por acerto)
- **Sistema de vidas**
- **Aumento gradual da dificuldade** (spawn mais rápido)
- **Tela de Game Over**
- **Botão de Começar o jogo**
- **Botão de Recomeçar**

# Tecnologias utilizadas

- **HTML5**
- **CSS3**
- **JavaScript**

# Como executar o projeto

- **Clone o repositório**

    git clone https://github.com/felipemasc31/jogo-letras

- **Abra o arquivo principal no navegador**

    index.html

# Como jogar

- As letras caem do topo da tela.
- Pressione a **tecla correspondente** para eliminá-la.
- Cada acerto vale **+10 pontos**.
- Se uma letra atingir o chão, você **perde 1 vida**.
- Com **0 vidas**, aparece a tela de **Game Over**.
- Clique em **Recomeçar** para tentar novamente.
